### Selected work

- **[Groundwork](https://github.com/Des-Sleigh/groundwork)** — flagship agent platform for AI-for-business research: an MCP server, a grounded research agent, and an orchestrator over a shared core. Every claim traces to a source; fetched content is treated as untrusted data. Measured: it ships **0% of hallucinations** vs naive RAG's 100%.
- **[llm-eval-harness](https://github.com/Des-Sleigh/llm-eval-harness)** — reproducible LLM evaluation: capability + benign-robustness tracks, LLM-as-judge plus deterministic graders, cost-aware tiered models, committed real results.
- **[NILPro](https://github.com/Des-Sleigh/nilpro)** — full-stack SaaS for NIL compliance across 51 U.S. jurisdictions ([thenilpro.com](https://thenilpro.com)). Next.js, TypeScript, Supabase, with database-layer security hardening.
- **[TrendMatch](https://github.com/Des-Sleigh/trendmatch-mvp)** — AI product-intelligence for TikTok creators ([live](https://trendmatch.netlify.app)). Next.js + Claude API + Supabase.
